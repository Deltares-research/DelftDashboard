install_delftdashboard.bat:
---------------------------
Creates the delftdashboard_git environment (existing enviroments with that name will be deleted!) and installs
Delft Dashboard and CHT dependencies directly from GitHub/Deltares

delftdashboard.bat:
-------------------
Runs Delft Dashboard. The first time it runs, it will ask for the location of the Delft Dashboard folder where
data will be stored.


update_delftdashboard.bat:
--------------------------
Updates the delftdashboard_git environment with latest versions of Delft Dashboard, CHT etc. on GitHub/Deltares
