Tiling
======

The Tiling toolbox (``tiling``) generates map tile pyramids for efficient
visualization of large model domains and bathymetry data.

Index Tiles
-----------

Generate index tiles that map raster pixels to computational grid cells.

Parameters
^^^^^^^^^^

Max Zoom (dropdown)
   Maximum zoom level for the tile pyramid. Higher zoom levels produce more
   tiles at finer resolution.

Actions
^^^^^^^

Generate Index Tiles
   Create the index tile pyramid for the current model domain.


TopoBathy Tiles
---------------

Generate topography/bathymetry tiles for background visualization.

Parameters
^^^^^^^^^^

Max Zoom (dropdown)
   Maximum zoom level for the tile pyramid.

Actions
^^^^^^^

Generate TopoBathy Tiles
   Create the topography/bathymetry tile pyramid for the current domain.
