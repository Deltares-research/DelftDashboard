Toolboxes
=========

Toolboxes provide specialized functionality that complements the model
editors in DelftDashboard. Some toolboxes are model-specific (such as the
Model Maker toolboxes), while others are general-purpose utilities that work
across all models.

Each toolbox appears as a tab in the side panel and is enabled or disabled
through the main configuration file ``delftdashboard.cfg``. Toolboxes that
are tied to a specific model are only shown when that model is active.
