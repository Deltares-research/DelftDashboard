Watersheds
==========

The Watersheds toolbox (``watersheds``) provides access to watershed
boundary datasets at various levels of detail. Users can browse watershed
boundaries on the map, select watersheds of interest, and export them.

Parameters
^^^^^^^^^^

Dataset (listbox)
   Select the watershed dataset (e.g. HydroBASINS).

Level (listbox)
   Select the watershed subdivision level. Lower levels give larger
   basins; higher levels give finer sub-basins.

Buffer (m)
   Buffer distance (in metres) to apply around selected watersheds when
   exporting.

Actions
^^^^^^^

Update
   Display watershed boundaries on the map for the selected dataset and
   level.

Save
   Export the selected watersheds to a file. Only enabled when at least
   one watershed is selected on the map.
