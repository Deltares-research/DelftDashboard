DelftDashboard Documentation
============================

DelftDashboard is a desktop application for setting up and visualizing coastal
and hydraulic numerical models. It provides a map-based GUI for creating model
grids, assigning bathymetry, defining boundary conditions, and managing
observation points for models like SFINCS, HurryWave, and Delft3D-FM.
